## 論文1: GeoTransolver (arXiv: 2512.20399v2)

**タイトル:** GeoTransolver: Learning Physics on Irregular Domains using Multi-scale Geometry Aware Physics Attention Transformer

**著者:** Corey Adams, Rishikesh Ranade, Ram Cherukuri, Sanjay Choudhry (NVIDIA)

### 概要

CAE（Computer-Aided Engineering）のためのAIサロゲートモデルとして、**マルチスケール幾何形状認識型Transformerアーキテクチャ**を提案。既存のTransolver（物理Attentionベースのモデル）を拡張し、標準的なAttention機構を**GALE（Geometry Aware Latent Embeddings）**に置き換えている。

### 核心技術: GALE Attention

- **物理状態スライスごとのSelf-Attention** と **幾何/グローバルコンテキストへのCross-Attention** を統合
- DoMINO由来の**マルチスケールBall Query**で、近傍（局所）と遠方（大域）の情報を複数半径で取得
- 幾何情報とグローバルパラメータを物理状態空間に射影し、全Transformerブロックで持続的にコンテキストとして注入（表現のドリフトを防止）
- 適応ゲーティングでSelf-AttentionとCross-Attentionの出力を動的にブレンド

### アーキテクチャの特徴

```
入力: 局所スライス(位置+特徴量) + 幾何形状 + グローバルパラメータ
  ↓
Context Projector: 幾何+グローバル情報からコンテキストベクトルを構築
  ↓
GALE Block × L層:
  ├─ スライス内Self-Attention
  ├─ コンテキストへのCross-Attention
  ├─ 適応ゲート(α)で混合
  └─ FFN + 残差接続
  ↓
出力ヘッド: per-slice正規化 → 予測
```

### ベンチマーク結果

| データセット | 対象 | 結果 |
|---|---|---|
| DrivAerML | 自動車外部空力(500形状) | 表面圧力L1誤差2.86%、抗力R²=0.996 |
| SHIFT-SUV | SUV外部空力(Estate/Fastback) | ほぼ全指標で最高精度 |
| SHIFT-Wing | 翼型遷音速空力(Ma=0.5/0.85) | 表面・体積指標で競争力のある精度 |

比較対象: Domino, AB-UPT, Transolver に対して概ね優位。

### 主な貢献

1. GALE Attention の提案（物理認識Self-Attention + 幾何Cross-Attention）
2. 幾何-コンテキスト射影の持続的注入でドリフト低減
3. マルチスケールBall Queryで局所精度と大域結合を両立
4. 3つのCAEベンチマークでSOTA級の精度

## 論文2: Benchmarking Framework (arXiv: 2507.10747v1)

**タイトル:** A Benchmarking Framework for AI models in Automotive Aerodynamics

**著者:** Kaustubh Tangsali, Rishikesh Ranade, Mohammad Amin Nabian, Alexey Kamenev, Peter Sharpe, Neil Ashton, Ram Cherukuri, Sanjay Choudhry (NVIDIA)

### 概要

自動車空力AIモデルの性能を**標準化された方法で比較評価**するためのベンチマーキングフレームワークを、NVIDIA PhysicsNeMo-CFD内に構築。DrivAerMLデータセットを用い、3つのAIモデル（DoMINO, X-MeshGraphNet, FIGConvNet）の精度・汎化性能を多角的に評価。

### フレームワークの構成

PhysicsNeMo-CFDが提供する評価ツール群:

1. **空力係数評価** — 抗力(Cd)・揚力(Cl)の予測精度、R²、Spearman相関、回帰プロット
2. **場の比較** — 表面コンター、体積コンター、中心線圧力分布（1D/2D/3Dマニフォルド対応）
3. **物理ベース指標** — 質量保存・運動量保存の残差（支配方程式の満足度）
4. **デザイントレンド分析** — 設計パラメータ変更に対する空力応答の傾向予測精度
5. **点群ベース検証** — STLからの均一サンプリングで、メッシュ非依存の公平な比較

### 評価対象モデル

| モデル | アーキテクチャ | 特徴 |
|---|---|---|
| **DoMINO** | DeepONet系ニューラルオペレータ | 点群から局所幾何符号化、表面+体積を同時予測 |
| **FIGConvNet** | Factorized Implicit Grid + U-Net | O(N²)計算量、2D再パラメータ化で効率的 |
| **X-MeshGraphNet** | マルチスケールGNN + Halo分割 | メッシュベースメッセージパッシング、大規模メッシュ対応 |

### 主な結果（DrivAerMLデータセット）

- **表面場**: DoMINOが面積加重L2誤差で最良（メッシュセル直接学習のため）
- **空力係数**: FIGConvNet・DoMINO > X-MeshGraphNet（積分量の損失項の有無が影響）
- **デザイントレンド**: DoMINO（Spearman Drag=0.99）が最も傾向を正確に捕捉
- **体積場**: DoMINOのみ評価（速度・圧力の誤差: 0.09〜0.20）

### 主な貢献

1. CFD視点からAIモデルを公平に比較する標準化フレームワーク
2. L2誤差だけでなく、力係数回帰、デザイントレンド、物理残差など多角的評価
3. 新モデル・新データセットの追加が容易なオープンソース設計
4. DrivAerMLでの3モデル比較による検証

## 2論文の関係性

両論文はNVIDIAの同一チームによる相互補完的な研究:

- **論文2（Benchmarking）** が「AIモデルをどう評価すべきか」の基盤を提供
- **論文1（GeoTransolver）** がその評価基盤上で新しいモデルアーキテクチャを提案し、既存手法（DoMINO含む）を上回る精度を実証

共通のキーワード: PhysicsNeMo, DrivAerML, サロゲートモデル, 自動車空力シミュレーション
